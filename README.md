# Ritesh Dubey - Personal Resume Website

This is a static HTML website built to showcase my resume, skills, professional experience, and GitHub projects. It is fully deployable via GitHub Pages.

## 📄 Overview

- Clean, responsive static resume site
- Resume downloadable in PDF format
- GitHub repositories dynamically fetched using GitHub API
- Simple HTML/CSS + JavaScript, no frameworks or dependencies

## 🧰 Tech Stack

- HTML5 & CSS3
- Vanilla JavaScript (fetch API)
- GitHub API (for repo listing)

## 🌐 Live Demo

Hosted via GitHub Pages:  
[https://ritesd.github.io/](https://ritesd.github.io/)

> Replace `username` and `<repo-name>` accordingly.

## 🚀 How to Use

1. Clone this repository or download the ZIP.
2. Replace or update the `RiteshDubey.pdf` if needed.
3. Commit and push to GitHub.
4. Enable GitHub Pages in **Settings > Pages** (use `main` branch, `/ (root)`).
5. Visit your new site using the GitHub Pages URL.
